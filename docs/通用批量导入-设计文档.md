# 通用批量导入审批框架 — 设计文档

## 1. 设计约束

- ❌ 数据库不存 JSON 数据
- ✅ **Table-Per-Hierarchy（TPH）**：通用列 + 类型独占列在同一张表，`ImportType` 列作鉴别器
- ✅ 策略模式处理类型差异（`IImportTypeHandler`）
- ✅ 审批通过后事件驱动创建业务实体

---

## 4. 策略模式核心接口

### 4.1 IImportTypeHandler

每种导入类型实现此接口，框架负责调用：

```csharp
// src/RBS.Core/Interfaces/Services/IImportTypeHandler.cs
public interface IImportTypeHandler
{
    string ImportType { get; }

    /// <summary>获取审批类型 ID</summary>
    Guid GetApprovalTypeId(Guid companyId);

    /// <summary>校验单行数据</summary>
    Task<List<ImportFieldError>> ValidateAsync(
        ImportBatchItem item, ImportValidationContext context, CancellationToken ct);

    /// <summary>审批通过后执行业务创建，返回创建数</summary>

}

public class ImportFieldError
{
    public string ErrorCode { get; set; } = "";
    public string Message { get; set; } = "";
    /// <summary>如何修正该错误，展示给用户</summary>
    public string? FixSuggestion { get; set; }
}

public class ImportValidationContext
{
    public Guid CompanyId { get; set; }
    public HashSet<string> ExistingKeys { get; set; } = new();
    public HashSet<string> BatchKeys { get; set; } = new();
    public Dictionary<string, object> CustomData { get; set; } = new();
}
```

## 5. 核心流程

### 5.1 SubmitAsync — 提交导入

```
POST /api/imports/submit
{
  "importType": "HousingUnit",
  "companyId": "guid",
  "fileName": "导入.xlsx",
  "items": [
    {
      "rowIndex": 3,
      "data": { ... }
    }
  ]
}
```

ImportService 处理逻辑：

```
① 根据 importType 获取 IImportTypeHandler
② 构建 ImportValidationContext
   - 查询现有 HousingUnits 建立 ExistingKeys（"座楼|楼层|房号"）
   - 查询 RoomTypes → CustomData["RoomTypeMap"]
   - 查询 FloorLevelBands、RoomPricingStandards → CustomData
③ 逐行校验（转换 data → 房源扩展实体 → 调用 handler.ValidateAsync）
④ 创建 ImportBatch（Status = PendingApproval）
⑤ 批量创建 ImportBatchItem（TPH：基础字段 + 房源独占字段）
⑥ 有有效行 → handler.GenerateSummaryAsync() → 创建 ApprovalRequest
⑦ CommitAsync → 返回结果
```

### 5.2 ExecuteApprovedImportAsync — 审批通过后执行

```
ApprovalCompletedEvent (TargetEntityType="Import")
  → ApprovalCompletedEventHandler
    → ImportService.ExecuteApprovedImportAsync(batchId)

① Load ImportBatch (含 Items)
② if Status != "PendingApproval" → return (幂等)
③ 根据 ImportType 获取 handler
④ handler.ExecuteAsync(batch):
   - 遍历 Items.Where(i => i.IsValid)
   - 向下转型为 ImportBatchItemHousingUnit
   - 二次检查唯一性：已存在的跳过
   - new HousingUnit() + AddAsync
⑤ batch.Status = "Approved"
⑥ CommitAsync
```

### 5.3 房源 Handler 完整实现

```csharp
public class HousingUnitImportHandler : IImportTypeHandler
{
    private readonly IUnitOfWork _uow;

    public string ImportType => "HousingUnit";

    // ==================== 常量 ====================

    private static readonly HashSet<string> ValidOrientations = new(StringComparer.OrdinalIgnoreCase)
    {
        "东", "南", "西", "北", "南北通透", "东南", "西南", "东北", "西北"
    };

    [GeneratedRegex(@"(\d+)")]
    private static partial Regex FloorNumberRegex();

    private static int ExtractFloorSortOrder(string floorName)
    {
        var m = FloorNumberRegex().Match(floorName);
        return m.Success ? int.Parse(m.Groups[1].Value) : 0;
    }

    // ==================== 审批类型 ====================

    public Guid GetApprovalTypeId(Guid companyId)
        => /* 查询 BATCH_IMPORT_ROOMS 的 ID */;

    // ==================== 逐行校验（9项校验） ====================

    public async Task<List<ImportFieldError>> ValidateAsync(
        ImportBatchItem item, ImportValidationContext context, CancellationToken ct)
    {
        var data = (ImportBatchItemHousingUnit)item;
        var errors = new List<ImportFieldError>();

        // ① 座楼名称 — 必填
        if (string.IsNullOrWhiteSpace(data.BuildingName))
            errors.Add(new() { ErrorCode = "ERR_REQUIRED_BUILDING", Message = "座楼名称为空", FixSuggestion = "请在「座楼名称」列填写座楼名称，如：A栋" });

        // ② 楼层 — 必填
        if (string.IsNullOrWhiteSpace(data.FloorName))
            errors.Add(new() { ErrorCode = "ERR_REQUIRED_FLOOR", Message = "楼层为空", FixSuggestion = "请在「楼层」列填写楼层，格式如：1层、2层" });

        // ③ 房号 — 必填
        if (string.IsNullOrWhiteSpace(data.UnitNo))
            errors.Add(new() { ErrorCode = "ERR_REQUIRED_UNIT", Message = "房号为空", FixSuggestion = "请在「房号」列填写房号，如：101" });

        // ④ 房型 — 必填
        if (string.IsNullOrWhiteSpace(data.RoomTypeName))
        {
            errors.Add(new() { ErrorCode = "ERR_REQUIRED_ROOMTYPE", Message = "房型为空", FixSuggestion = "请在「房型」列从下拉列表中选择系统现有房型" });
        }
        else
        {
            // ⑤ 房型名称 → 匹配系统房型
            var roomTypeMap = context.CustomData["RoomTypeMap"] as Dictionary<string, Guid>;
            if (roomTypeMap != null && !roomTypeMap.TryGetValue(data.RoomTypeName, out _))
            {
                var available = string.Join("、", roomTypeMap.Keys.Take(10));
                errors.Add(new()
                {
                    ErrorCode = "ERR_INVALID_ROOMTYPE",
                    Message = $"房型「{data.RoomTypeName}」不存在，系统可用房型：{available}",
                    FixSuggestion = "请从 Sheet2 参考数据中选择系统现有房型名称填入"
                });
            }
        }

        // ⑥ 面积 — 大于0
        if (data.Area.HasValue && data.Area.Value <= 0)
            errors.Add(new() { ErrorCode = "ERR_INVALID_AREA", Message = $"面积必须是大于0的数字，当前值：{data.Area}", FixSuggestion = "请检查「面积」列，填写正数，如：85.5" });

        // ⑦ 朝向 — 仅允许预设值
        if (!string.IsNullOrEmpty(data.Orientation) && !ValidOrientations.Contains(data.Orientation))
            errors.Add(new()
            {
                ErrorCode = "ERR_INVALID_ORIENTATION",
                Message = $"朝向「{data.Orientation}」不合法，仅允许：{string.Join("、", ValidOrientations)}",
                FixSuggestion = "请在「朝向」列从下拉列表中选择，可选值：东、南、西、北、南北通透、东南、西南、东北、西北"
            });

        // ⑧ 房源唯一性 — (BuildingName + FloorName + UnitNo)
        var key = $"{data.BuildingName}|{data.FloorName}|{data.UnitNo}";
        if (context.BatchKeys.Contains(key))
            errors.Add(new() { ErrorCode = "ERR_DUPLICATE_UNIT", Message = $"该房源（{data.BuildingName}-{data.FloorName}-{data.UnitNo}）在本次导入中重复", FixSuggestion = "请删除重复行，每条座楼+楼层+房号组合只能出现一次" });
        else if (context.ExistingKeys.Contains(key))
            errors.Add(new() { ErrorCode = "ERR_DUPLICATE_UNIT", Message = $"该房源（{data.BuildingName}-{data.FloorName}-{data.UnitNo}）已存在", FixSuggestion = "该房源已存在系统中，如需修改请删除对应行后重导，或修改座楼/楼层/房号避免重复" });

        // ⑨ 完整编码唯一性 — FullCode 全局唯一
        var generatedFullCode = $"{data.BuildingName}-{data.FloorName}-{data.UnitNo}";
        if (context.BatchKeys.Contains($"FC:{generatedFullCode}"))
            errors.Add(new() { ErrorCode = "ERR_DUPLICATE_FULLCODE", Message = $"完整编码「{generatedFullCode}」在本次导入中重复", FixSuggestion = "请修改座楼/楼层/房号组合避免完整编码重复" });
        else if (context.CustomData.TryGetValue("ExistingFullCodes", out var fcObj)
                 && ((HashSet<string>)fcObj).Contains(generatedFullCode))
            errors.Add(new() { ErrorCode = "ERR_DUPLICATE_FULLCODE", Message = $"完整编码「{generatedFullCode}」已存在", FixSuggestion = "该完整编码已存在，请修改座楼/楼层/房号组合避免重复" });

        return errors;
    }

    // ==================== 审批通过后执行（含金额自动计算） ====================

    public async Task<int> ExecuteAsync(ImportBatch batch, CancellationToken ct)
    {
        var allBands = await _uow.FloorLevelBands.GetAllAsync(ct);
        var companyPricing = (await _uow.RoomPricingStandards.GetAllAsync(ct))
            .Where(p => p.CompanyId == batch.CompanyId).ToList();
        var existingUnits = await _uow.HousingUnits.GetAllAsync(ct);
        var existingKeys = new HashSet<string>(
            existingUnits.Select(u => $"{u.BuildingName}|{u.FloorName}|{u.UnitNo}"),
            StringComparer.OrdinalIgnoreCase);

        int created = 0;

        foreach (var item in batch.Items.OfType<ImportBatchItemHousingUnit>().Where(i => i.IsValid))
        {
            // 二次检查唯一性（防止并发/重导时重复创建）
            var key = $"{item.BuildingName}|{item.FloorName}|{item.UnitNo}";
            if (existingKeys.Contains(key)) continue;

            // 金额自动计算
            var floorSortOrder = ExtractFloorSortOrder(item.FloorName ?? "");
            decimal? baseRent = null;
            string? priceWarning = null;

            if (floorSortOrder > 0 && item.RoomTypeId.HasValue)
            {
                var band = allBands.OrderBy(b => b.MinLevel)
                    .FirstOrDefault(b => b.MinLevel <= floorSortOrder && b.MaxLevel >= floorSortOrder);

                if (band != null)
                {
                    var pricing = companyPricing
                        .FirstOrDefault(p => p.RoomTypeId == item.RoomTypeId.Value && p.FloorLevelBandId == band.Id);
                    if (pricing != null)
                        baseRent = pricing.RentAmount;
                    else
                        priceWarning = $"房型「{item.RoomTypeName}」在{band.Name}无匹配定价标准，租金未设置";
                }
                else
                {
                    priceWarning = $"楼层「{item.FloorName}」超出所有楼层分级范围，租金未设置";
                }
            }
            else if (floorSortOrder == 0)
            {
                priceWarning = $"楼层「{item.FloorName}」格式无法识别（未能提取楼层数字），租金未设置";
            }

            // 创建 HousingUnit
            var unit = new HousingUnit(item.BuildingName!, item.FloorName!,
                floorSortOrder, item.UnitNo!, batch.CompanyId);
            unit.SetBuildingInfo(item.BuildingCode, item.BuildingAddress);
            unit.SetFullCode($"{item.BuildingName}-{item.FloorName}-{item.UnitNo}");
            unit.UpdateDetails(item.RoomTypeId, item.Area, item.Orientation, baseRent);

            await _uow.HousingUnits.AddAsync(unit, ct);
            existingKeys.Add(key); // 批次内防重
            created++;
        }

        return created;
    }

    // ==================== 审批摘要 ====================

    public async Task<string> GenerateSummaryAsync(ImportBatch batch, CancellationToken ct)
    {
        var items = batch.Items.OfType<ImportBatchItemHousingUnit>().Where(i => i.IsValid).ToList();
        return JsonSerializer.Serialize(new {
            文件名 = batch.FileName,
            总行数 = batch.TotalRows,
            有效行 = items.Count,
            覆盖座楼 = items.Select(i => i.BuildingName).Distinct().ToList(),
            覆盖房型 = items.Where(i => i.RoomTypeName != null).Select(i => i.RoomTypeName).Distinct().ToList(),
        });
    }
}
```

### 7.2 取消审批

```
POST /api/approvals/{id}/cancel
Content-Type: application/json

Request: { "reason": "提交人撤回" }

Response: ApprovalRequestDto
```

**后端逻辑**：
1. 校验状态为 `Pending`
2. 校验当前用户是提交人（第一条 `Submitted` 记录的操作人）
3. 调用 `entity.Cancel(reason)` 将状态改为 `Cancelled`
4. 如果关联导入批次（`TargetEntityType == "Import"`），同步更新批次状态为 `Cancelled`

```csharp
StatusLabel = batch.Status switch

    "Cancelled" => "已撤回",

```

前端也保留客户端映射函数作为兜底。

## 11. 前端操作反馈与过渡效果

### 8.1 全页面加载遮罩

提交和下载等异步操作使用 Element Plus `v-loading` 全页面遮罩，防止用户误操作中断请求：

```html
<div v-loading="pageLoading" 
     :element-loading-text="loadingText"
     element-loading-background="rgba(0,0,0,0.45)">
```

| 操作 | pageLoading | loadingText |
|------|-------------|-------------|
| 下载模板 | `true` | 「正在生成模板...」 |
| 提交审批 | `true` | 「正在提交审批，请勿关闭页面...」 |
| 完成 | `false` | `''` |

### 8.2 按钮加载态

按钮本身也使用 Element Plus `:loading` 属性，点击后显示旋转加载图标并禁用：

| 操作 | loading 绑定 | 文字切换 |
|------|-------------|----------|
| 下载模板 | `:loading="downloading"` | 「正在生成模板...」 |
| 提交审批 | `:loading="submitting"` | 「正在提交审批...」 |

页面遮罩 + 按钮加载双重保障，确保操作不会被中断。

### 8.3 页面切换动画

三个页面状态（upload → preview → result）使用 Vue `<Transition>` 组件：

```html
<Transition name="page-fade" mode="out-in">
  <div key="upload" v-if="pageStatus === 'upload'">...</div>
  <div key="preview" v-if="pageStatus === 'preview'">...</div>
  <div key="result" v-if="pageStatus === 'result'">...</div>
</Transition>
```

CSS 定义：

```css
.page-fade-enter-active,
.page-fade-leave-active {
  transition: opacity 0.25s ease, transform 0.25s ease;
}
.page-fade-enter-from {
  opacity: 0;
  transform: translateY(10px);
}
.page-fade-leave-to {
  opacity: 0;
  transform: translateY(-10px);
}
```

- `mode="out-in"`：先离开再进入，避免两个状态同时存在导致的布局抖动
- 持续时间 0.25s：足够产生平滑感，又不至于拖沓

## 12. 重导兼容策略

| 场景 | 处理 |
|------|------|
| 审批驳回后修正重导 | 创建新批次，旧批次标记 Rejected 可追溯 |
| 部分行已审批通过 | `ExecuteAsync` 中二次检查唯一性，已有行自动跳过 |
| 同一文件再次提交 | 允许，每次新批次 + 新审批流程 |
| 已导入的行修改后重导 | 座楼+楼层+房号不变 → 跳过；变了 → 作为新房源创建 |
